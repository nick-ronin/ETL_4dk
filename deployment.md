# Развёртывание проекта

## Описание

Данный документ описывает процесс запуска и развёртывания сервиса обработки данных.

Проект представляет собой API-сервис на FastAPI, который принимает CSV/XLSX файлы, выполняет их обработку, нормализацию полей, поиск коллизий и формирует результат обработки.

На текущем этапе система работает только с файлами и не выполняет загрузку данных в базу данных. Результатом работы являются обработанные данные в формате JSON и Excel-отчёты с найденными совпадениями.

## Требования к окружению

Для запуска проекта необходимо:

- Python версии 3.10 или выше;
- установленный пакетный менеджер pip;
- доступ к файловой системе для создания временных файлов и сохранения результатов обработки.

Основные зависимости проекта находятся в файле: requirements.txt


В проекте используются следующие библиотеки:

- **FastAPI** — создание API и обработка запросов;
- **Uvicorn** — запуск сервера приложения;
- **Pandas** — обработка и анализ табличных данных;
- **OpenPyXL** — работа с Excel-файлами;
- дополнительные библиотеки для выполнения операций очистки, нормализации и проверки данных.
