# Logger

Модуль предназначен для ведения журнала работы приложения. Он записывает информацию о ходе выполнения ETL-процесса, возникающих ошибках и других служебных сообщениях. Логи сохраняются в файл и одновременно выводятся в консоль.

---

# Назначение

Модуль выполняет следующие задачи:

- настройка системы логирования;
- запись сообщений в файл журнала;
- вывод сообщений в консоль во время работы приложения;
- создание функции для записи сообщений одновременно в логгер и дополнительный текстовый файл.

---

# Процесс работы

После импорта модуля:

1. Настраивается система логирования.
2. Устанавливается уровень логирования `INFO`.
3. Определяется формат сообщений журнала.
4. Создаются обработчики:
   - запись сообщений в файл `output/log/app.log`;
   - вывод сообщений в консоль.
5. Создаётся объект `logger`, который используется другими модулями проекта.
6. При вызове `get_log_writer()` возвращается функция для записи сообщений в журнал.

---

# Основные методы

### `get_log_writer()`

Создаёт функцию для записи сообщений в журнал.

### Параметры

```python
get_log_writer(log_path: str = None)
```

### Возвращает

Функцию `write(msg)`, которая:

- записывает сообщение через стандартный логгер;
- при наличии `log_path` дополнительно сохраняет сообщение в указанный текстовый файл.

---

# Формат результата

После вызова функции записи сообщение:

- отображается в консоли;
- сохраняется в файл:

```text
output/log/app.log
```

Если указан `log_path`, сообщение также записывается в дополнительный файл.

Формат записи:

```text
2026-07-10 15:21:34 [INFO] Загружен файл companies.xlsx
```

---

# Пример работы

```python
from logger.logger import get_log_writer

log = get_log_writer()

log("Начало обработки файла")
log("Нормализация завершена")
```

Результат в журнале:

```text
2026-07-10 15:21:34 [INFO] Начало обработки файла
2026-07-10 15:21:36 [INFO] Нормализация завершена
```

Пример с дополнительным файлом:

```python
log = get_log_writer("process.log")

log("Начало обработки")
```

Сообщение будет записано:

- в `output/log/app.log`;
- в `process.log`.
---

# Возможные ошибки

| Ошибка | Причина |
|---------|----------|
| `FileNotFoundError` | Папка `output/log` отсутствует и не может быть создана. |
| `PermissionError` | Недостаточно прав для записи файла журнала. |
| `OSError` | Ошибка файловой системы при создании или записи логов. |
| `UnicodeEncodeError` | Ошибка кодировки при записи сообщения в файл. |
