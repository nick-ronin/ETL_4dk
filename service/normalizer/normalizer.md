# Нормализация данных

Модуль нормализации предназначен для приведения данных из различных источников к единому формату. Он очищает значения полей, устраняет различия в написании и подготавливает информацию для сопоставления, поиска дубликатов и дальнейшей обработки.

---

# Назначение

Модуль выполняет нормализацию следующих данных:

- телефонных номеров;
- ИНН;
- адресов электронной почты;
- кратких названий организаций;
- полных названий организаций;
- городов;
- адресов.

После обработки одинаковые данные, записанные в разных форматах, приводятся к единому виду, что позволяет корректно сравнивать записи из различных источников.

---

# Входные данные

Модуль принимает:

- строковые значения отдельных полей (`str`);
- объект `pandas.DataFrame` для обработки названий организаций.

Поддерживаемые поля:

| Поле | Описание |
|------|----------|
| `phone` | Телефон |
| `inn` | ИНН |
| `email` | Адрес электронной почты |
| `short_name` | Краткое название организации |
| `full_name` | Полное название организации |
| `city` | Город |
| `address` | Адрес |

---

# Процесс работы

## Нормализация телефона (`clean_phone`)

Функция:

- удаляет лишние символы;
- поддерживает несколько номеров в одной ячейке;
- заменяет первую цифру `8` на `+7`;
- приводит номера к формату `+7XXXXXXXXXX`;
- сохраняет существующий знак `+`;
- удаляет пустые значения.

Пример

```python
clean_phone("8 (999) 123-45-67")
```

Результат

```text
+79991234567
```

---

## Нормализация ИНН (`clean_inn`)

Функция:

- удаляет все символы кроме цифр;
- проверяет длину ИНН (10 или 12 цифр);
- проверяет корректность значения;
- возвращает сообщение об ошибке при неверном формате.

Пример

```python
clean_inn("77-07-089-389")
```

Результат

```text
7707089389
```

---

## Нормализация Email (`clean_email`)

Функция:

- приводит адреса к нижнему регистру;
- удаляет лишние пробелы;
- поддерживает несколько email в одной строке;
- оставляет только корректные адреса;
- удаляет повторяющиеся значения.

Пример

```python
clean_email("TEST@MAIL.RU; info@mail.ru")
```

Результат

```text
test@mail.ru, info@mail.ru
```

---

## Очистка краткого названия (`clean_short_name`)

Функция:

- удаляет организационно-правовую форму (ООО, АО, ПАО и др.);
- удаляет кавычки;
- удаляет текст после первой запятой;
- удаляет лишние пробелы;
- приводит название к верхнему регистру.

Пример

```python
clean_short_name('ООО "Ромашка"')
```

Результат

```text
РОМАШКА
```

---

## Очистка полного названия (`clean_full_name`)

Функция:

- расшифровывает сокращённые организационно-правовые формы;
- удаляет кавычки;
- удаляет лишние символы;
- приводит название к верхнему регистру.

Пример

```python
clean_full_name('ООО "Ромашка"')
```

Результат

```text
ОБЩЕСТВО С ОГРАНИЧЕННОЙ ОТВЕТСТВЕННОСТЬЮ РОМАШКА
```

---

## Нормализация названий организаций (`normalize_company_names`)

Функция выполняет обработку DataFrame.

Если колонка `short_name` отсутствует, она создаётся на основе `full_name`.

Если отсутствует `full_name`, создаётся пустая колонка.

В результате обе колонки приводятся к единому стандарту.

---

## Извлечение города (`clean_city`)

Функция извлекает название города из адресной строки.

Поддерживаются конструкции:

- `г. Москва`
- `город Москва`

Пример

```python
clean_city("123456, г. Москва, ул. Ленина, д. 5")
```

Результат

```text
Москва
```

---

## Очистка названия города (`normalize_city_text`)

Если значение начинается с:

- `г.`
- `город`

то функция возвращает только название города.

Пример

```python
normalize_city_text("г. Санкт-Петербург")
```

Результат

```text
Санкт-Петербург
```

---

## Нормализация адреса (`clean_address`)

Функция:

- удаляет индекс;
- удаляет название города (если оно передано отдельным параметром);
- удаляет конструкции `г.` и `город`;
- удаляет обозначения улицы;
- нормализует обозначения корпуса и строения;
- удаляет слово `дом`;
- приводит адрес к единому формату.

Пример

```python
clean_address(
    "123456, г. Москва, ул. Ленина, д. 10",
    city="Москва"
)
```

Результат

```text
Ленина, 10
```

# Основные методы

| Метод | Назначение |
|--------|------------|
| `clean_phone()` | Нормализация телефонов |
| `clean_inn()` | Проверка и очистка ИНН |
| `clean_email()` | Очистка адресов электронной почты |
| `clean_short_name()` | Очистка краткого названия организации |
| `clean_full_name()` | Очистка полного названия организации |
| `normalize_company_names()` | Нормализация названий организаций в DataFrame |
| `clean_city()` | Извлечение города из адреса |
| `normalize_city_text()` | Очистка названия города |
| `clean_address()` | Нормализация адресов |

---

# Формат результата

Большинство функций возвращают очищенное строковое значение.

Метод `normalize_company_names()` возвращает обновлённый `DataFrame`.

```python
df = normalize_company_names(df)
```

После обработки будут присутствовать колонки:

- `short_name`;
- `full_name`.

---

# Пример работы

```python
import pandas as pd

from service.normalizer.normalizer import (
    normalize_company_names,
    clean_phone,
    clean_email,
    clean_inn,
    clean_city,
    clean_address
)

df = pd.DataFrame({
    "full_name": ['ООО "Ромашка"'],
    "phone": ["8 (999) 123-45-67"],
    "email": ["TEST@MAIL.RU"],
    "inn": ["77-07-089-389"],
    "address": ["123456, г. Москва, ул. Ленина, д. 10"]
})

df = normalize_company_names(df)

df["phone"] = df["phone"].apply(clean_phone)
df["email"] = df["email"].apply(clean_email)
df["inn"] = df["inn"].apply(clean_inn)

df["city"] = df["address"].apply(clean_city)

df["address"] = df.apply(
    lambda row: clean_address(row["address"], row["city"]),
    axis=1
)
```

Полученные данные используются модулем дедупликации, модулем оценки качества данных и экспортируются в итоговые отчёты.

# Возможные ошибки

| Ситуация | Результат |
|----------|-----------|
| `NaN` или `None` | Возвращается исходное значение |
| Некорректный ИНН | `"Неверный формат ИНН"` |
| Некорректный email | `None` |
| Некорректный телефон | Возвращается очищенное значение либо `None` |
| Отсутствуют `short_name` или `full_name` | Создаются пустые колонки |
| Город в адресе не найден | `clean_city()` возвращает пустую строку |
| Пустой DataFrame | DataFrame возвращается без изменений |
