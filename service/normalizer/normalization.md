# Нормализация данных

Модуль нормализации предназначен для приведения данных из различных источников к единому формату. Он очищает значения полей, устраняет различия в написании и подготавливает информацию для сопоставления, поиска дубликатов и дальнейшей обработки.

---

## Назначение

Модуль выполняет нормализацию следующих данных:

- телефонных номеров;
- ИНН;
- адресов электронной почты;
- кратких названий компаний;
- полных названий компаний;
- адресов.

После обработки одинаковые данные, записанные в разных форматах, приводятся к единому виду, что позволяет корректно сравнивать записи из разных источников.

---

## Входные данные

Модуль принимает:

- строковые значения отдельных полей (`str`);
- объект `pandas.DataFrame` для обработки названий компаний.

Поддерживаемые поля:

- `phone`
- `inn`
- `email`
- `short_name`
- `full_name`
- `address`

---

## Процесс работы

### Очистка телефона (`clean_phone`)

- удаление лишних символов;
- разделение нескольких номеров;
- преобразование номера в формат `+7XXXXXXXXXX`;
- замена префикса `8` на `+7`.

**Пример**

```python
clean_phone("8 (999) 123-45-67")
```

Результат:

```text
+79991234567
```

---

### Очистка ИНН (`clean_inn`)

- удаление всех символов, кроме цифр;
- проверка длины;
- проверка корректности значения.

**Пример**

```python
clean_inn("77-07-089-389")
```

Результат:

```text
7707089389
```

---

### Очистка Email (`clean_email`)

- перевод в нижний регистр;
- удаление пробелов;
- поиск корректных email;
- удаление повторяющихся адресов.

**Пример**

```python
clean_email("TEST@MAIL.RU; info@mail.ru")
```

Результат:

```text
test@mail.ru, info@mail.ru
```

---

### Очистка краткого названия (`clean_short_name`)

- удаление организационно-правовой формы;
- удаление кавычек;
- удаление описания после запятой;
- перевод в верхний регистр.

**Пример**

```python
clean_short_name('ООО "Ромашка"')
```

Результат:

```text
РОМАШКА
```

---

### Очистка полного названия (`clean_full_name`)

- расшифровка сокращённой организационно-правовой формы;
- удаление кавычек;
- удаление лишних символов;
- перевод в верхний регистр.

**Пример**

```python
clean_full_name('ООО "Ромашка"')
```

Результат:

```text
ОБЩЕСТВО С ОГРАНИЧЕННОЙ ОТВЕТСТВЕННОСТЬЮ РОМАШКА
```

---

### Очистка адреса (`clean_address`)

- удаление лишних пробелов;
- нормализация обозначений улиц;
- приведение корпуса и строения к единому формату;
- удаление обозначения дома.

**Пример**

```python
clean_address("ул. Воронцовская, д. 6 строение 1")
```

Результат:

```text
Воронцовская, 6 стр1
```

---

## Основные методы

| Метод | Назначение |
|--------|------------|
| `clean_phone()` | Нормализация телефонов |
| `clean_inn()` | Проверка и очистка ИНН |
| `clean_email()` | Очистка адресов электронной почты |
| `clean_short_name()` | Нормализация краткого названия компании |
| `clean_full_name()` | Нормализация полного названия компании |
| `normalize_company_names()` | Нормализация названий компаний в DataFrame |
| `clean_address()` | Нормализация адресов |

---

## Формат результата

Каждый метод возвращает очищенное значение соответствующего поля.

Метод `normalize_company_names()` возвращает обновлённый `DataFrame`.

```python
df = normalize_company_names(df)
```

После выполнения в таблице будут нормализованы колонки:

- `short_name`
- `full_name`

---

## Пример работы

Исходные данные:

```python
import pandas as pd

df = pd.DataFrame({
    "short_name": ['ООО "Ромашка"'],
    "phone": ["8 (999) 123-45-67"],
    "email": ["TEST@MAIL.RU"],
    "address": ["ул. Воронцовская, д. 6 строение 1"]
})

df = normalize_company_names(df)

df["phone"] = df["phone"].apply(clean_phone)
df["email"] = df["email"].apply(clean_email)
df["address"] = df["address"].apply(clean_address)
```

Результат:

| short_name | phone | email | address |
|------------|--------|--------|---------|
| РОМАШКА | +79991234567 | test@mail.ru | Воронцовская, 6 стр1 |

---

## Использование в проекте

Модуль используется после загрузки данных и сопоставления колонок.

```python
df = normalize_company_names(df)
df["inn"] = df["inn"].apply(clean_inn)
df["phone"] = df["phone"].apply(clean_phone)
df["email"] = df["email"].apply(clean_email)
df["address"] = df["address"].apply(clean_address)
```

Полученные данные используются для поиска дубликатов и формирования итогового результата.

---

## Возможные ошибки

| Ситуация | Результат |
|----------|-----------|
| `NaN` или `None` | Возвращается исходное значение |
| Некорректный ИНН | `"Неверный формат ИНН"` |
| Некорректный Email | `None` |
| Отсутствуют `short_name` и `full_name` | Создаются пустые значения |
